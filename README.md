Requirements:
For pose recognition:
pip install yolo
pip install scikit-learn

For voice recognition:
New system environment variable for token
pip install sounddevice
pip install whisper

Instrctions:
Pressing "q" terminates the webcam and the process.

Run camera + audio together:
python run_realtime.py



 -------------------- Front-end --------------------
 Technology:
  1. Node.js
 - Download url: https://nodejs.org/en/download
 - Select (React), then select (Typescript and React compilent)
 - Optional: You can check your version by typing: node -v

 2. Npm
 - Navigate to the poseapp folder
 - Run: npm install
 - Optional: You can check your version by typing: npm -v

 3. Electron
 - Navigate to the frontend folder
 - In the terminal, type: "npm install electron --save-dev"
 
 How to start poseapp:
 1. Navigate to the poseapp folder, example (...\solita_project\frontend\poseapp>)
 2. Run: "npm run dev"

 How to start electron:
 1. Navigate to the frontend folder, example (...\solita_project\frontend)
 2. Run: "npm run dev"
